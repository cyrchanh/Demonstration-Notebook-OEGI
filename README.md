# Demo Notebook

## Quick Start

1. **Open the notebook**
   Open `notebook.ipynb` in [Google Colab](https://colab.research.google.com/) or any Jupyter-compatible environment.

2. **Upload supporting files**
   Once the runtime is connected, upload all files from this directory into the notebook's working directory. In Colab, you can do this via the 📁 file browser in the left sidebar — just drag and drop everything in.

   Key files to include:
   - `traces/` — JSONL trace logs used by the replay visualisation
   - Any `.py` modules or config files referenced by the notebook

3. **Run all cells**
   Go to **Runtime → Run all** (or `Ctrl+F9`) and let it execute top to bottom. Each section will print its own outputs, charts, and interactive widgets inline.

That's it — no extra installs or setup needed beyond what the notebook handles itself.
